# [`sebimoe/php-fpm-with-extensions`](https://hub.docker.com/r/sebimoe/php-fpm-with-extensions) docker image

A simple image based on the original [`php:fpm`](https://hub.docker.com/_/php) docker image, with multiple commonly used extensions installed using [`mlocati/docker-php-extension-installer`](https://github.com/mlocati/docker-php-extension-installer).

See `Dockerfile` for details. See [`php`](https://hub.docker.com/_/php) image documentation for usage.
